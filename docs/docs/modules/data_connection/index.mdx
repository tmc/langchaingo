# 数据连接

:::info
[概念指南](https://python.langchain.com/docs/modules/data_connection/indexing)
:::

许多 LLM 应用程序需要不属于模型训练集的用户特定数据。LangChain 为您提供了通过以下方式加载、转换、存储和查询数据的构建块：

- [文档加载器](/docs/modules/data_connection/document_loaders/)：从多种不同来源加载文档
- [文本分割器](/docs/modules/data_connection/text_splitters/)：分割文档和文本
- [向量存储](/docs/modules/data_connection/vectorstores/)：存储和搜索嵌入数据
- [检索器](/docs/modules/data_connection/retrievers/)：查询您的数据
