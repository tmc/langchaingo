# 模型 I/O

使用语言模型通常可以分为三个步骤。

 - 首先构建模型的输入。在 langchain 中，这是使用 [prompts 包](/docs/modules/model_io/prompts/) 完成的。
 - 接下来必须将输入发送到模型。使用 [model 包](/docs/modules/model_io//) 可以轻松做到这一点。
 - 在许多情况下，需要从模型输出中提取信息。使用 [output parser 包](/docs/modules/model_io/output_parser) 可以简单地做到这一点。
