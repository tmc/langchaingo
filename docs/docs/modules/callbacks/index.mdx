# Callbacks

In addition to being able to create your own, LangchainGo ships with a variety of callback handlers that you can use to extend the functionality of your LangChainGo application. This includes the following:

* [OpenTelemetry Callbacks Handler](./opentelemetry): This callback handler allows you to trace the calls made to your LangChainGo application using OpenTelemetry.
