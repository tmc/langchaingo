# 快速入门：LangChainGo 与 OpenAI

通过运行您的第一个 LangChainGo 和 [OpenAI](https://openai.com/) 程序来开始。OpenAI 的 GPT 模型以其卓越的性能和广泛的功能而闻名。

## 前提条件

1.  **OpenAI API 密钥**：在 [OpenAI](https://openai.com/) 上注册并获取您的 API 密钥。
2.  **Go**：[下载并安装 Go](https://go.dev/doc/install)。

## 设置

在与 OpenAI API 交互之前，您需要将 API 密钥设置为环境变量。

### Linux/macOS (bash/zsh)
```bash
export OPENAI_API_KEY="此处填写您的OpenAI_API密钥"
```

### Windows (命令提示符)
```cmd
set OPENAI_API_KEY=此处填写您的OpenAI_API密钥
```

### Windows (PowerShell)
```powershell
$env:OPENAI_API_KEY="此处填写您的OpenAI_API密钥"
```

要进行永久设置，请将环境变量添加到您 shell 的配置文件（`~/.bashrc`、`~/.zshrc` 等）或 Windows 的系统环境变量中。

## 步骤

1.  **设置您的 OpenAI API 密钥**：按照上面的设置说明配置您的 API 密钥。

2.  **运行示例**：执行以下命令：

```shell
go run github.com/tmc/langchaingo/examples/openai-completion-example@main
```

您应该会看到类似以下的输出：

```shell
第一个登上月球的人是尼尔
```
