# 未完成的快速入门：使用聊天模型

聊天模型是语言模型的一种变体。
虽然聊天模型底层也使用语言模型，但它们暴露的接口略有不同。
它们并非暴露“文本输入，文本输出”的 API，而是暴露一个以“聊天消息”作为输入和输出的接口。

## 安装

首先，使用以下命令安装 LangChain：

```bash
go get github.com/tmc/langchaingo
```
